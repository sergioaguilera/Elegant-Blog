    QGIS es una aplicación que facilita el trabajo de mapeo, en especial cuando se trata de posicionar los puntos en tu sistema de Geo Posicionamiento.

    Es una plataforma libre, que esta disponible para windows, gnu/linux y mac, entre sus principales bondades es que tiene una gran intereaccion con Postresql y demasiado facil e intuitiva, ademas nos permite la creación de vectores, sin mencionar que pues bueno... es software libre.

    Cuenta con un sin fin de repositorios para facilitarnos la vida, desde repositorios para usar los mapas de Google, hasta repositorios de OpenStreetMap

    Pueden saber mas de QGIS desde la pagina official.

    En ubuntu, su instalación es muy sencilla solo hace falta abrir una terminal en modo de superusuario y teclear

    sudo apt-get install qgis python-qgis

    Sin embargo te instalara la versión de lanzamiento, que si bien, es funcional, cuenta con algunos bugs poco molestos a la hora de trabajar con un proyecto grande, (por ejemplo a la hora de cargar un proyecto este truena si tenia integrada alguna capa de mapas cargada via plugins).
